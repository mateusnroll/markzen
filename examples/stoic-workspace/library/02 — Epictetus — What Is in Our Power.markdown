# What Is in Our Power?

> “Of things some are in our power, and others are not.”

*Epictetus, Encheiridion I, George Long translation*

## The control inventory

The sentence is short, but using it well requires a second question: what response is actually mine now? The table in `04 — Commonplace Index.md` keeps examples; this page keeps the method.

- Within my authorship
    - the judgment I rehearse
    - the request I make
    - the care I bring to the next action
- Outside my authorship
    - another person's verdict
    - yesterday's outcome
    - the timing of an interruption

## A decision routine

```text
observe the impression
name what is mine
release the demanded outcome
choose the next honorable action
```

1. Describe the event without verdict.
2. Separate action from outcome.
3. Choose a response consistent with **wisdom**, **justice**, **courage**, and **temperance**.
4. Review without self-dramatizing.

## Practice queue

- [x] Write one neutral description of a difficult event.
- [ ] Name the judgment added to it.
    - [ ] Ask whether that judgment is necessary.
    - [ ] Replace prediction with preparation.
- [ ] Do the next useful thing.

The method is not indifference. It is disciplined attention to the part of a situation where choice can become character.
